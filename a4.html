Movie API Documentation

Root URL: http://localhost:2000

1. Retrieve All Movies

GET /

Retrieves all records from the Movies table.

Sample Response

[{"Name": "Inception", "ReleaseDate": "2010-07-16", "Rating": 8.8, "Director": "Christopher Nolan", "Genre": "Action, Adventure, Sci-Fi"}]

2. Retrieve All Movies Sorted by Rating

GET /movies/sorted-by-rating

Retrieves all movies sorted from the highest rating to the lowest.

Sample Response

[{"Name": "The Shawshank Redemption", "ReleaseDate": "1994-10-14", "Rating": 9.3, "Director": "Frank Darabont", "Genre": "Drama"}]

3. Search Movies by Genre

GET /movies/search

Searches for movies by genre.

Request Parameters

genre (query parameter)

Sample Response

[{"Name": "The Dark Knight", "ReleaseDate": "2008-07-18", "Rating": 9.0, "Director": "Christopher Nolan", "Genre": "Action, Crime, Drama"}]

4. Retrieve Specific Movie Information

GET /:Name

Retrieves information of a specific movie by its name.

Request Parameters

Name (path parameter)

Sample Response

{"Name": "Interstellar", "ReleaseDate": "2014-11-07", "Rating": 8.6, "Director": "Christopher Nolan", "Genre": "Adventure, Drama, Sci-Fi"}

5. Insert a New Movie Record

POST /:Name

Inserts a new record into the Movies table.

Request Body Example

{
        "Name": "Dunkirk",
        "ReleaseDate": "2017-07-21",
        "Rating": 7.9,
        "Director": "Christopher Nolan",
        "Genre": "Action, Drama, History"
    }

Sample Response

{"Success": "Successful: Record was added!."}

6. Update an Existing Movie Record

PUT /:Name

Updates an existing movie record by its name.

Request Body Example

{
        "Name": "Dunkirk",
        "ReleaseDate": "2017-07-21",
        "Rating": 8.0,
        "Director": "Christopher Nolan",
        "Genre": "Action, Drama, History"
    }

Sample Response

{"Success": "Successful: Record was updated!."}

7. Delete a Movie Record

DELETE /:Name

Deletes a movie record by its name.

Request Parameters

Name (path parameter)

Sample Response

{"Success": "Successful: Record was deleted!"}
